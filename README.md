NHENTAI注射器，将中文翻译注入到 N 站体内。

参考[EhSyringe](https://github.com/EhTagTranslation/EhSyringe)并使用EhSyringe词库制作的NHENTAI翻译脚本。

这一针打下去你就会说中文了。
